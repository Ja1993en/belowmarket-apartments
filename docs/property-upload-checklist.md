# Property Upload Checklist

Use one CSV row per floor plan. Repeat the same property name for every floor plan that belongs to that property.

Before importing:

- Property gallery photos go in `Property Photo URLs`.
- Floor plan photos go in `Floor Plan Photo URLs`.
- Multiple photo URLs in one cell should be separated with `|`.
- `Available Units` must be the real available unit count.
- Use `0`, blank, `Unavailable`, or `Not Currently Available` for unavailable floor plans.
- Put the property-level special and floorplan-level special in `Special` only when it applies to that row.
- Use plain numbers for rent and fees, such as `1795` and `158`.
- Keep new properties in `Draft` or `Pending Review` until photos, pricing, specials, and availability are verified.

After importing:

- Open the public property page and confirm gallery photos.
- Confirm every floor plan has the correct image, rent, fees, special, and availability.
- Search for the city and property name on the property search page.
- Only make the property `Live` after the public page matches the source.
